# The Heritage Market
This is platform on which students of your local school can buy/sell/trade things (like textbooks and calculators) and services (such as tutoring) between eachother.

## Features
This project has multiple features planned:
- Messenger (to be made)
- Listings displays (in development right now)
- Account management (in development right now)

## Download
To make a local working copy of this project:
1. Clone this repository into a local folder.
2. Download PostgreSQL 18 & Node.js (v22.17.0 is what I used).
3. Run npm install once each in the frontend & backend folders in the folder you cloned to.
4. Start the PostgreSQL server on port 3000
5. Start node server on port 3000 by running ```node server``` in the backend folder terminal.

Should be it.
